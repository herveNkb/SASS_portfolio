# Histoire

Création d'un portfolio inspiré de https://developerfolio.js.org/

Avec comme techno HTML et SASS

## Pédagogie
Ce portfolio a été réalisé lors d'un projet sur la plateforme STUDI, dans le cadre de la formation de développeur web et web mobile. L'objectif était de créer un portfolio personnel pour présenter mes compétences et mes projets.
